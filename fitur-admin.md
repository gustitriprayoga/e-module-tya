1. Research Command Center (Dashboard Utama)
   Pusat pemantauan statistik penelitian secara real-time untuk kebutuhan data Bab IV.

Analytics Overview: Grafik rata-rata skor Pre-test vs Post-test.

WPM Tracker: Pantauan kecepatan baca rata-rata mahasiswa (Words Per Minute).

N-Gain Summary: Persentase efektivitas media berdasarkan kategori (High, Medium, Low).

Active Samples: Status 23 mahasiswa sampel (siapa yang sudah selesai, siapa yang belum).

2. Dynamic Module Builder (ADDIE - Design & Development)
   Fitur Page-Builder untuk menyusun 14 sesi pertemuan secara fleksibel tanpa menyentuh kode.

Module Management: CRUD (Create, Read, Update, Delete) mata kuliah (Reading II).

Session Organizer: Mengatur urutan 14 sesi (Drag & Drop).

Interactive Block System: Menambahkan blok konten pada setiap sesi:

PBL Intro Block: Untuk memicu masalah (Problem-Based Learning).

Reading Block: Input teks bacaan dengan pengaturan Target WPM.

Vocabulary Block: Menautkan kata-kata sulit ke database kosakata.

Quiz Block: Latihan pemahaman di tengah materi.

3. Instrument & Evaluation Management
   Mengelola alat ukur penelitian berdasarkan Blue Print Test.

Question Bank: Bank soal untuk Pre-test dan Post-test.

Indicator Mapping: Menandai soal berdasarkan 5 indikator (Main Idea, Inference, dll).

Test Configuration: Mengatur durasi pengerjaan, skor minimal, dan kunci jawaban.

Automated Scoring: Sistem penilaian otomatis yang langsung terhubung ke tabel analitik.

4. Vocabulary Vault (Target 3.500 Kata)
   Database terpusat untuk penguasaan kosakata mahasiswa.

Bulk Import: Unggah ribuan kata sekaligus menggunakan file Excel/CSV.

Glossary Management: Mengatur definisi, contoh kalimat, dan kategori kata.

Word Tracking: Melihat kata-kata mana yang paling sering gagal dipahami oleh mahasiswa.

5. Research Analytics & Data Export (Bab IV Support)
   Fitur paling krusial untuk memproses data penelitian Anda.

Individual Progress: Laporan detail per mahasiswa (Skor T1, T2, dan hasil N-Gain).

Class Distribution: Grafik sebaran kemampuan mahasiswa sebelum dan sesudah intervensi.

Export Data: Unduh data mentah ke format Excel (XLSX) atau PDF untuk dilampirkan di lampiran skripsi/tesis Anda.

6. User & Campus SSO Monitor
   Manajemen pengguna yang terintegrasi dengan sistem kampus.

SSO Sync Log: Memantau mahasiswa yang login menggunakan akun SAINS Universitas Pahlawan.

Role Management: Mengatur hak akses (siapa yang menjadi Dosen, Admin, atau Mahasiswa).

Student Activity: Melihat log aktivitas mahasiswa (kapan terakhir belajar).
